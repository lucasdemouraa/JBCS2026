A body of knowledge represented in propositional logic uses logical connectives of implication (→), which represents the conditional; conjunction (∧), which represents logical AND; disjunction (∨), which represents logical OR; and negation (¬), which represents the logical operator NOT. Let P be the following set of formulas of propositional logic:

1. a → ¬b  
2. b ∧ a  
3. ¬b ∨ b  

Let Q be the following set of formulas of propositional logic:

4. a ∨ b  
5. b → ¬a  

And let R be the formula:

6. ¬b → a  

See the truth table for these formulas.

[Table displaying truth values for propositions a and b, and corresponding truth values for formulas 1 through 6. The table has columns labeled “a”, “b”, “a → ¬b”, “b ∧ a”, “¬b ∨ b”, “a ∨ b”, “b → ¬a”, and “¬b → a”, with rows that provide the truth values, True or False, for each formula according to the combination of truth values for a and b.]

It is known that each row in the truth table corresponds to an assignment of truth values to the propositional symbols a and b, and each column corresponds to the evaluation of the formula for that assignment. Consider the following definitions:

(i) A formula is a tautology if and only if, for every assignment of truth values, its evaluation is true.

(ii) An assignment of truth values satisfies a set of formulas if and only if, for every formula in the set, the evaluation is true.

(iii) A set of formulas is satisfiable if and only if there exists an assignment of truth values that satisfies the set. Otherwise, it is unsatisfiable.

(iv) A formula is a logical consequence of a set of formulas if and only if, for every truth-value assignment that satisfies the set, that assignment also satisfies the formula.

Based on the information presented, answer the following items:

a) Is there any tautology among formulas 1 to 6? Justify your answer. (2.5 points)

b) Is there any satisfiable set, P or Q? Justify your answer. (2.5 points)

c) Is there any unsatisfiable set, P or Q? Justify your answer. (2.5 points)

d) Is formula 6 a logical consequence of Q? Justify your answer. (2.5 points)
